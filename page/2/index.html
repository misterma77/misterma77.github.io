创建一个字符串

传统的方式

通过new关键字 来创建的是 一个对象

字符串中的方法

1.replace(参数1,参数2):  替换
    1.想替换哪个字符串，就用哪个字符串来调用
    2.参数1是被替换的目标， 参数2是替换的新内容
    3.会将替换后的字符串，整体返回
2.substring(参数1,参数2) : 提取 介于两个下标之间的 字符串    包头不包尾  (和数组slice相似)
    参数1： 开始下标      参数2： 结束下标
3.substr(参数1,参数2): 可在字符串中 从开始位置截取指定长度的字符串 (和数组splice相似)
    参数1： 开始位置，  参数2 ： 指定长度
4.split()方法:  用于把一个字符串 分割成字符串数组(本质上就是个数组，只是存的都是字符串罢了)
    参数1 ：  分割的依据 
    参数2：(可选参数)分割后 返回的长度 
    (超出实际大小，按照实际大小算,小于实际大小的话，按照指定长度返回，超出的不返回)
5.fromCharCode( ) 接受一个UniCode编码，返回对应字符串
6.charAt(位置)    返回指定位置的字符
7. .length    返回字符串的长度
8.charCodeAt(位置）    返回指定位置字符的Unicode编码
9.slice(参数1,参数2)    提取指定范围的字符串
    参数1：开始范围    参数2：结束范围    包头不包尾
10.indexOf()    返回指定字符的对应下标（首先出现的），如果没有，则返回-1
11.lastIndexOf()    返回指定字符最后出现的位置
12.toLowerCase()    字符串转换为小写
   toUpperCase()    字符串转换为大写

Posted on 2020-01-15

数组：

数组是存储1个或多个数据的容器，是一组内存空间，
通常用来批量处理数据
这组内存空间的名字叫做数组名
数组对自身储存的数组元素没有限制，不论数量类型
通过数组下标（从0开始）来管理和操作数组
数组名.length    表示数组的长度，也是储存元素的个数；因为下标从0开始，所以数组长度
    减1表示最后一个元素下标

创建数组：
创建一个数组

访问数组:

数组方法：

  1.push( )：将一个元素添加到数组的最后一位
      数组名.push( 添加的元素 )
  2.unshift( )：将一个元素添加到数组的第一位
      数组名.unshift( 添加的元素 ）
  3.pop( )：删除数组的最后一个元素
      数组名.pop(  )
  4.shift( )：删除数组的第一个元素
      数组名.shift(  )
  5.slice(开始,结束)：截取数组的指定范围，并返回
      截取的是开始到结束的前一位（复制新内容）
  6.splice(开始位置，修改长度）：修改原数据，不复制
      删除：只规定开始位置与修改长度
      替换：规定了开始位置，修改长度和新内容
      插入：新内容的个数多于修改的长度
  7.join( )：将数组中的元素转换为字符串，拼接并返回
  8.sort( )：数组排序    从小到大
  9.reverse( )：倒序排列
10.cancat：将后者拼接到前者的后面，结合成新的数组
11.instancdof：运算，判断它是不是一个数组，返回true/false

变量的声明： var s;

变量的定义： var s = 10;

全局变量:不在函数中声明或定义.(之前的写法);
作用范围：是当前整个文件. 任何地方都可以获取并修改它

函数

Posted on 2020-01-15

函数的整体语法：

  1. 定义函数：

    声明函数的关键字 函数名(参数)

    重要事项：

    1.函数只定义，不调用的话，是不执行 的
    2.函数在哪里调用，就在哪里执行
    3.函数调用的次数，不限
    形式参数列表（简称形参）：自定义名称,不需要加var 就可以声明内存空间
    实际参数（简称实参）： 是向函数内部，传递的数据
    在调用函数的过程中，实参是具体的数据，形参只是一块空白的内存空间；
    实参向形参传参这个过程， 就是一个值拷贝的过程。
    形参负责存储实参拷贝的值，并在函数中使用
    但函数执行结束后，形参这块内存被回收，下次函数再被调用时，重新分配
    给形参空间;
    实参给形参赋值， 是按照实参()中，书写的顺序的来的
    函数的四种形态：

  2. 无参无返回值

  3. 有参无返回值

  4. 无参有返回值

  5. 有参有返回值

    定义一个函数，求出两个数的乘积，并返回，以供后续功能使用
    函数的返回值 return
    1. 需要写在return关键字后，可以是一个值，也可以是一个表达式计算出来的值。
    2. return在一个函数中，理论上可以出现很多个，但是只有一个会执行
    3. return被触发后，当前函数立刻结束，它下面的代码不再执行。
      注 1. 函数在哪里调用， 返回值就去到了哪里
      2.有返回值的情况，一定需要新建变量来接受

      函数嵌套：

      不允许嵌套定义， 嵌套的是调用
      比较两个数的大小 形参的命名：
      (1)在定义函数的小括号中自定义，符合命名规则即可。
      (2)一旦定义过，在当前函数内，必须按照定义好的名字使用

基本运算符

可以操作常量也可以操作并保持变量
加号两边都是number类型的，运行加法
加号任意一边是string类型，那么就会运行拼接（即将加号两边都变成字符串类型进行拼接）
NaN 特殊标识符，表示该结果不是数字，它自身不等自身

复合运算符

自增运算符(++增量为1)

(1) ++在前，参与当前运算
(2) ++在后，不参与当前运算，但所有运算执行结束后再自增

自减运算符(--减量为1)

(1) --在前，参与当前运算
(2) --在后，不参与当前运算，但所有运算执行结束后再自减

关系运算符

== 等于 数值形同即可，即使数据类型不同也会返回true
=== 全等于 数值和数据类型不同会返回false
！ 取反运算

逻辑运算符

（1）逻辑 与 &&
语法：最后结果 = 表达式1 && 表达2
逻辑与： 两个表达式 ，都为真，最后结果才是true,有一个是假的。结果就是false。 一假即假

（2）逻辑或 ||
语法是和 逻辑与一致的
注意点：逻辑与 是一假即假。 逻辑或，只要有一个表达式
为真，最后结果即为真。 一真即真

（3）逻辑非 取反运算 !
只是把true或false 相互转换
true 或false , 有一些其他的表达方式 。
例如： YES or NO， 正确 或 错误，真或者假，0 或 1，非0即为真

三元运算符(三目运算符) 又被称为 问号冒号表达式

语法： 最后结果 = 表达式 ？ 值1 : 值2；
判断表达式的结果，如果为真，就返回 值1, 否则返回 值2
