javascript

Posted on 2020-01-14 Edited on 2020-01-15

1. 什么是javascript?

javascript 是一种具有面向对象能力的解释性语言，它是基于对象和事件驱动并且具有相对安全性的客户端脚本语言。

2. javascript特点：

(1) 松散型: 它的变量不具有一个明确的类型
(2) 对象属性：javascript中把属性映射为任意属性
(3) 继承属性：JavaScript中的继承是基于原型的

3. javascript 核心

(1) 核心ECMAscript 5.0
(2) 文档对象模型 DOM
(3) 浏览器对象模型 BOM

4. javascript 能做什么？

(1)嵌入动态文本于html页面
(2)对浏览器事件作出响应
(3)读写html 元素
(4)在数据被提交到服务器前验证数据
(5)检测访客浏览器信息
(6)控制cookies 包括创建和修改
