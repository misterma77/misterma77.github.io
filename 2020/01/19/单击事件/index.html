给元素添加 单击事件

onclick: 鼠标的单击事件
方式一：  获取元素，在js中给元素绑定
    备注： 直接赋值使用，不需要函数名
    xx.onclick=function(){
        需要执行的任务
    }
    方式二： 定义好函数， 在html标签中 绑定函数
    function XXX(){
        需要执行的任务
    }
