事件监听

事件监听: addEventListener(参数1，参数2，参数3)

参数1 ： 事件类型
参数2 ： 触发事件后执行的函数
参数3 ： 布尔值   默认值是 false 事件冒泡  ；   true 事件捕获

普通事件绑定和事件监听的区别：

1.普通事件绑定只能给元素相同类型的时间绑定一个，如果绑定第二个会覆盖上
  事件监听可以给同一个元素的相同类型事件绑定多个
2.事件监听可以控制事件的传播方式（事件冒泡或事件捕获）
3.事件监听可以通过删除的方法，来取消事件，而普通绑定只能通过null来取消
4.事件监听可以对动态生成的元素（未来元素）生效，而普通事件不可以

删除事件监听 removeEventListener()三个参数

注意事项：1.相同元素调用
         2.相同事件名称
         3.相同函数名(匿名函数不行，需要全局函数)
         4.相同的执行阶段

事件委托：

利用事件冒泡机制指定一个事件处理程序，来管理某一类型的所有时间
即利用事件冒泡的原理，把事件加到父级上，触发执行效果
优点:1.只在内存中开辟了一块空间，节省资源的同时，减少了dom操作，提高性能
     2.对于新添加的元素也会有之前的事件

BOM

Bom主要用于管理窗口和窗口之间的通讯，其核心对象为windowwindow对象所有浏览器都支持 window 对象。它代表浏览器的窗口。所有全局 JavaScript 对象，函数和变量自动成为 window 对象的成员。全局变量是 window 对象的属性。全局函数是 window 对象的方法。甚至（HTML DOM 的）document 对象也是 window 对象属性窗口尺寸两个属性可用用于确定浏览器窗口的尺寸。这两个属性都以像素返回尺寸：

* window.innerHeight - 浏览器窗口的内高度（以像素计）
* window.innerWidth - 浏览器窗口的内宽度（以像素计）

浏览器窗口（浏览器视口）不包括工具栏和滚动条。对于 Internet Explorer 8, 7, 6, 5：

* document.documentElement.clientHeight
* document.documentElement.clientWidth

或

* document.body.clientHeight
* document.body.clientWidth

* window.open() - 打开新窗口
* window.close() - 关闭当前窗口
* window.moveTo() -移动当前窗口
* window.resizeTo() -重新调整当前窗口

事件流

事件流：

从页面中接受事件的顺序

事件捕获：

不太具体的DOM节点应该更早接收到事件，而最具体的节点应该最后接收到事件、
从上到下，从不具体到具体

事件冒泡：

IE的事件流叫做事件冒泡
即事件开始时，有最具体的元素接受，然后逐级向上传播到较为不具体的节点
从下到上，从具体到不具体

三个阶段：事件捕获阶段，处于目标阶段，事件冒泡阶段

事件的目标在捕获阶段不会接受到事件，下一阶段处于目标阶段，事件在目标上发生，
    在事件处理中被看成冒泡阶段的一部分

键盘事件 

1.onkeydown:  用户按下键位的时候触发
2.onkeyup :   用户抬起键位时
3.onkeypress: 用户按下该键位 再抬起时触发

事件类型 

1.onload : 当页面(html,图片之类)加载完成后，执行的方法该方法用于，包裹所有
    内联js代码
    1). 解决js获取html标签时，标签还没有加载的情况
    2). 先加载页面，再加载功能。 能够提高用户体验
    window.onload = function(){
        这里存放 所有内联的js代码
        以保证,先加载页面,再加载js功能              
    }     
2.onblur： 失去焦点
    ipt1.onblur = function(){
        console.log("我失去了焦点");
    }
3.onfocus: 获得焦点
    ipt1.onfocus = function(){
        console.log("我获得了焦点");
    }
4.onselect: 选中文本时触发
    ipt1.onselect = function(){
        console.log("我被选中了");
    }
5.onchange: 在内容发生改变是触发
    ipt1.onchange = function(){
        console.log("我被改了");
    }
6.onreset: 重置时触发
    var res = document.getElementsByClassName("re")[0];
    res.onreset = function (){
        console.log("我被重置了");
    }
7.ondblclick: 双击时触发
    var but = document.getElementsByClassName("btn")[0];
    but.ondblclick = function(){
        console.log("我被双击了");
    }
8.onmouseover:鼠标移入时触发
9.onmouseout:鼠标移出时触发
10.onmousedown：用户在指定元素身上按下时触发
   除非情景需要按下和抬起各执行一个功能，否则不使用
11.onmouseup：抬起时触发

Date

创建一个Date对象

概念： Date对象是用来处理时间和日期，内置了一系列获取和设置日期和时间的方法
使用：
    1.每次在使用它之前，都需要配合new关键字， 来生成一个Date对象，
    2.然后通过Date对象，再去调用各种方法
    Date获取的是当前计算机的本地时间
    var date = new Date();
时间格式转换：toLocaleString()： 可把时间对象 转换为字符串(我们需要的格式)
getTime(): 返回的是  1970年1月1日距今的 毫秒数
在下面的练习中， 我们将计算出，从1970/1/1至今 过去多少年

—————— 设置 时间和日期 ——————–

1.setTime(): 以毫秒数 设置Date对象
    date.setTime(93599945322);
2.setFullYear():  设置  年  月  日    备注：  月份 取值  0 ~ 11
    date.setFullYear(2019,7,24);
3.setMonth() : 设置   月  日 毫秒
    date.setMonth(7,24);
4.setHours() : 设置 时  分 秒 毫秒
    date.setHours(12,0,0);
5.setMinutes():  分 秒  毫秒
    date.setMinutes();

——————- 获取时间 和 日期 ———————

1.获取年份：console.log(date.getFullYear());
2.获取月份（0~11）console.log(date.getMonth());
3.获取一个月中的第几天：console.log(date.getDate());
4.获取一周中的第几天：console.log(date.getDay());
5.获取小时：console.log(date.getHours());
6.获取分钟：console.log(date.getMinutes());
7.获取秒数：console.log(date.getSeconds());

字符串

创建一个字符串

传统的方式

通过new关键字 来创建的是 一个对象

字符串中的方法

1.replace(参数1,参数2):  替换
    1.想替换哪个字符串，就用哪个字符串来调用
    2.参数1是被替换的目标， 参数2是替换的新内容
    3.会将替换后的字符串，整体返回
2.substring(参数1,参数2) : 提取 介于两个下标之间的 字符串    包头不包尾  (和数组slice相似)
    参数1： 开始下标      参数2： 结束下标
3.substr(参数1,参数2): 可在字符串中 从开始位置截取指定长度的字符串 (和数组splice相似)
    参数1： 开始位置，  参数2 ： 指定长度
4.split()方法:  用于把一个字符串 分割成字符串数组(本质上就是个数组，只是存的都是字符串罢了)
    参数1 ：  分割的依据 
    参数2：(可选参数)分割后 返回的长度 
    (超出实际大小，按照实际大小算,小于实际大小的话，按照指定长度返回，超出的不返回)
5.fromCharCode( ) 接受一个UniCode编码，返回对应字符串
6.charAt(位置)    返回指定位置的字符
7. .length    返回字符串的长度
8.charCodeAt(位置）    返回指定位置字符的Unicode编码
9.slice(参数1,参数2)    提取指定范围的字符串
    参数1：开始范围    参数2：结束范围    包头不包尾
10.indexOf()    返回指定字符的对应下标（首先出现的），如果没有，则返回-1
11.lastIndexOf()    返回指定字符最后出现的位置
12.toLowerCase()    字符串转换为小写
   toUpperCase()    字符串转换为大写
